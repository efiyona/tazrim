# מדריך מערכת "התזרים" — ידע מוצר מפורט לעוזר **תזרי**

- **version:** 2.0.0  
- **last_updated:** 2026-04-15  
- **שפה ממשק:** עברית (RTL)

מסמך זה מתאר את האפליקציה כפי שמיושמת בקוד הפרויקט. כשמפנים משתמש לעמוד, להשתמש בפורמט: `[[PAGE:/נתיב/מלא|טקסט קצר לכפתור]]` (נתיב יחסי מ־root האתר, מתחיל ב־`/`).

---

## 1. מושגי יסוד

| מושג | הסבר |
|------|------|
| **בית (Home)** | ישות משפחתית/משותפת: כל משתמשים מחוברים לאותו `home_id` רואים את אותן פעולות, קטגוריות והגדרות בית. |
| **משתמש** | משתמש עם חשבון (התחברות). יש `first_name`, `nickname`, וכו'. פעולות נשמרות עם `user_id` של מי שיצר/עדכן. |
| **קטגוריה** | תווית להכנסה או להוצאה (למשל: קניות, משכורת). לכל קטגוריית הוצאה אפשר להגדיר **תקציב חודשי** (`budget_limit`). |
| **פעולה (עסקה)** | רשומת הכנסה או הוצאה: סכום, תיאור, קטגוריה, **תאריך ביצוע** (`transaction_date`). |
| **פעולה קבועה (חוזרת)** | תבנית שמייצרת אוטומטית פעולות בכל חודש (לפי יום בחודש). מנוהלות ב"ניהול הבית". |
| **חודש תצוגה** | החודש/שנה שנבחרים בדף הבית/דוחות נשמרים ב־session (`view_month`, `view_year`) ונשארים גם כשעוברים בין דפים (אלא אם משנים ב־URL). |

## 2. ניווט כללי באפליקציה

### 2.1 תפריט תחתון (בר ניווט)

- **ראשי** → `/index.php` — דף הבית. כפתור **פלוס** מרחף ליד הבר: פותח מודאל **הוספת פעולה** (`add-transaction-modal`).
- **דוחות** → `/pages/reports.php`
- **קניות** → `/pages/shopping.php` — כפתור פלוס פותח מודאל פריט קניות (`add-shopping-item-modal`) כשהדף טעון.
- **הגדרות** (תפריט מורחב):
  - **ניהול הבית** → `/pages/settings/manage_home.php`
  - **החשבון שלי** → `/pages/settings/user_profile.php`

### 2.2 שורת עליון (Header)

- **אזור פרופיל ושם הבית** — בצד אחד.
- **צ'אט (תזרי)** — כפתור עם אייקון שיחה (`aiChatLauncher`): פותח מודאל מסך מלא של העוזר תזרי (מודול `app/features/ai_chat`).
- **התראות** — פעמון; רשימת התראות (למשל אחרי הזרקות פעולות קבועות).
- **התנתקות** → `/logout.php`

### 2.3 דיווח מהיר

- כפתור צף **"דיווח מהיר"** (באג / רעיון) — פותח מודאל שליחת דיווח בפורמט קצר.

### 2.4 תקנון וגישה

- משתמשים חייבים לאשר **תקנון** עדכני (`accept_tos.php`) לפי גרסה בקוד.

## 3. דף הבית — `/index.php`

### 3.1 ניווט חודש

- בראש הדף: **חודש קודם** | **החודש הנבחר** | **חודש הבא** + כפתור **"היום"** כשלא רואים את החודש הקלנדרי.
- פרמטרים ב־URL: `?m=MM&y=YYYY` (למשל `index.php?m=3&y=2026`).

### 3.2 כרטיסי KPI (לפי החודש הנבחר)

- **הוצאות** (חודש) — עם כפתור **פירוט** לפי סוג `expense`.
- **הכנסות** (חודש) — עם **פירוט** לפי סוג `income`.
- **יתרה בחשבון** — מוצגת רק אם הוגדרה **יתרה התחלתית** לבית בניהול הבית; מחושבת מיתרה התחלתית + פעולות עתידיות ועבר.

### 3.3 רשימות פעולות

- **פעולות ממתינות** — תאריך הפעולה **אחרי היום** (עתידי לחודש הנוכחי בתצוגה). ניתן לערוך/למחוק.
- **פעולות אחרונות** — תאריך עד היום. "עוד…" לטעינת עוד רשומות (AJAX).
- לחיצה על פעולה פותחת מודאל **עריכה**.

### 3.4 קטגוריות והתקציב

- גריד קטגוריות הוצאה עם סכום שנוצל בחודש ומדידת תקציב (אם הוגדר).

### 3.5 רענון חלקי

- אחרי פעולות מסוימות, אזור הליבה עלול להתעדכן ב־AJAX (`home_dashboard_core.php`) בלי רענון מלא של הדף.

## 4. דוחות ותובנות — `/pages/reports.php`

- אותו **ניווט חודש** כמו בדף הבית (כולל `?m=&y=`).
- **KPI**: סהכ הכנסות, סהכ הוצאות, מאזן החודש, ממוצע הוצאה יומית.
- **גרף עוגה** — התפלגות הוצאות לפי קטגוריה.
- **מדי תקציב** — קטגוריות עם `budget_limit` והוצאה בפועל בחודש.

## 5. רשימת קניות — `/pages/shopping.php`

### 5.1 אשף ראשוני (אם אין חנויות)

- שלב 1–2: בחירת **חנויות** (סופר, ירקן, פארם וכו' או שם מותאם).
- שמירה יוצרת רשומות ב־`shopping_categories` ומאפשרת את מסך הקניות.

### 5.2 מסך קניות רגיל

- טאבים לפי **חנות**; רשימות פריטים לפי חנות.
- ניהול חנויות (הוספה/עריכה/מחיקה) דרך מודאלים בעמוד.

---

## 6. ניהול הבית — `/pages/settings/manage_home.php`

סקשנים עיקריים (לפי המימוש):

- **קוד הצטרפות** — קוד להזמנת חברי בית; קישור לשליחה בוואטסאפ עם טקסט מוכן.
- **פרטי הבית** — שם בית, **יתרה התחלתית** (מוצפנת במסד), וכו'.
- **קטגוריות** — הוספה/עריכה/מחיקה (הוצאה/הכנסה), תקציב לקטגוריית הוצאה.
- **פעולות קבועות** — רשימת תבניות חודשיות (הוצאה/הכנסה), יום בחודש, סכום, קטגוריה; הפעלה/כיבוי.
- **חנויות קניות** — רשימת חנויות לטאבי הקניות (שם, אייקון, סדר).

## 11. טיפים למענה למשתמש (לתזרי)

- **"איפה מוסיפים פעולה?"** — דף הבית, כפתור הפלוס בבר התחתון או מודאל הוספה.
- **"איך מזיזים חודש?"** — שבבי החודש בראש דף הבית או דוחות; או `?m=&y=`.
- **"איך מזמינים בן לבית?"** — ניהול הבית → קוד הצטרפות, או `register.php?join_code=...`.
- **"למה אין לי יתרה?"** — אם לא הוגדרה יתרה התחלתית בניהול הבית, כרטיס היתרה עלול לא להופיע.
- **"למה הופיעה פעולה לבד?"** — כנראה **פעולה קבועה** שהוזרקה אוטומטית; בדוק ב־ניהול הבית → פעולות קבועות ובהתראות.
